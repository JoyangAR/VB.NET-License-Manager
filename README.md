VB.NET-License-Manager
======================

This is an extended port of the C# License Manager, that was introduced in the dotnetpro article 'Persilscheine für Ihre Software' (issue 10/2013, German).

This port shows, how Portable.Licensing can be used in Visual Basic .NET.

Thanks to Thorsten Weiß (weiss.thorsten@gmx.de) for this work!
